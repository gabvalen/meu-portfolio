# Portfólio Profissional

Portfólio profissional desenvolvido em HTML5 e CSS3 como parte do Projeto 1 – Portfólio Profissional Interativo.

Este repositório contém o código do meu Portfólio Profissional Interativo, desenvolvido como parte do Projeto 1 – Portfólio Profissional Interativo. O objetivo do projeto é construir uma presença digital pública, organizada e funcional, apresentando minha trajetória, formação e projetos iniciais na área de tecnologia e dados.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Git
- GitHub Pages

## Estrutura do Portfólio

O portfólio foi estruturado com foco em clareza, semântica e facilidade de navegação, contemplando:

- Apresentação pessoal
- Formação acadêmica
- Vitrine de projetos
- Integração com GitHub e LinkedIn
- Canal de contato
- Layout responsivo

## Justificativa Técnica

O projeto foi implementado como um website estático utilizando exclusivamente HTML5 e CSS3, conforme proposto no desafio. As escolhas técnicas priorizaram organização do conteúdo, estrutura semântica adequada e simplicidade visual, buscando uma apresentação profissional e de fácil compreensão.

O controle de versão foi realizado com Git, registrando a evolução do projeto por meio de commits associados a ajustes estruturais, refinamento de conteúdo e consolidação da entrega final. A publicação foi realizada via GitHub Pages, garantindo acesso público e alinhamento com o objetivo de visibilidade profissional estabelecido no projeto.
